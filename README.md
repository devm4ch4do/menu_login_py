# Menu de Login em Python

Este projeto é uma aplicação simples de terminal desenvolvida em Python para praticar conceitos básicos de lógica de programação, estruturas de dados e interação com o usuário.

## Visão Geral

O sistema oferece um menu interativo com opções para:
- cadastrar um novo usuário;
- realizar login;
- acessar uma área logada;
- sair da aplicação.

Os dados dos usuários são mantidos em memória durante a execução do programa.

## Funcionalidades

- Cadastro de usuários com nome, e-mail e senha;
- Validação de confirmação de senha;
- Verificação de e-mails duplicados;
- Autenticação de login;
- Exibição de dados do usuário após o login;
- Opção de logout.

## Requisitos

- Python 3.8 ou superior;
- Terminal ou prompt de comando.

## Como executar

1. Acesse a pasta do projeto:
   ```bash
   cd /workspaces/menu_login_py
   ```

2. Execute o arquivo principal:
   ```bash
   python3 login/login.py
   ```
